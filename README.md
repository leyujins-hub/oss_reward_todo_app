# 보상형 성취 Todo List 앱

## 1. 프로젝트 소개

이 프로젝트는 대학생이 과제와 학습 목표를 지속적으로 관리할 수 있도록 만든 보상형 Todo List 앱입니다. 사용자는 할 일을 등록하고 완료할 때마다 경험치(EXP)를 얻으며, 기린 캐릭터의 성장과 통계 화면을 통해 자신의 성취를 시각적으로 확인할 수 있습니다.

## 2. 개발 배경

기존 Todo List는 할 일을 기록하고 체크하는 기능에 집중되어 있어 사용자가 지속적으로 앱을 사용하도록 만드는 동기부여가 약합니다. 저희는 과제 완료, 집중 시간, 통계, 캐릭터 성장 요소를 결합하여 사용자의 학습 습관 형성과 자기주도적 학습을 돕고자 했습니다.

## 3. 주요 기능

- 할 일 추가, 수정, 삭제, 완료 처리
- 마감일, 중요도, 카테고리 설정
- 중요도 높은 과제 카드 강조
- 과제 완료 시 EXP 증가
- 기린 캐릭터 레벨 및 진행도 표시
- 과제별 집중 시간 타이머
- 주간, 월간, 연간 통계
- 업적 및 성장 분석
- 학교 관련 빠른 링크 제공
- 날짜별 퀘스트 분류: 미룬 퀘스트, 오늘 퀘스트, 내일 퀘스트

## 4. Adaptiveness

사용자의 할 일 상태, 중요도, 마감일, 완료 여부에 따라 UI와 피드백이 달라집니다.

- 마감일이 지난 미완료 과제는 미룬 퀘스트로 분류
- 오늘 날짜의 과제는 오늘 퀘스트로 분류
- 내일 날짜의 과제는 내일 퀘스트로 분류
- 중요도가 높은 과제는 붉은 카드로 강조
- 과제를 완료하면 EXP와 성장 통계에 반영
- 집중 시간에 따라 학습 상태가 시각적으로 드러남
