# Draft Screens

이 폴더는 팀원들이 생성형 AI 또는 Figma AI를 활용하여 얻은 초기 화면 초안을 정리한 공간입니다.

초안 화면은 최종 구현본과 일부 차이가 있으며, 최종 구현 과정에서 발표 기간, 구현 가능성, UI 일관성, 배포 안정성을 기준으로 기능을 통합·수정했습니다.

## 폴더 구성

| 폴더명 | 팀원 | 담당 초안 |
|---|---|---|
| ANN | 안혜원 | 기본 Todo List 화면, 기린 캐릭터 성장 시스템 |
| MIN | 김민지 | 오늘의 명언, 통계 페이지, 업적 및 성장 분석 |
| DO | 김도연 | 집중 시간 타이머, 학교 빠른 링크 기능 |
