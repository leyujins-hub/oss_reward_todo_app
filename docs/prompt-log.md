# 생성형 AI 활용 기록

## 1. 안혜원

### 사용 프롬프트
- 다기능 Todo List 애플리케이션 개발
- 제목, 마감일, 중요도, 카테고리, 상태 속성을 가진 Todo List 구현
- 캐릭터 성장 시스템과 EXP 시스템을 결합한 학습 관리 앱 구현
- 중요도 높은 과제를 빨간색 카드로 강조
- 완료 시 EXP +10 증가
- Lv.1부터 Lv.5까지 기린 캐릭터가 성장하는 구조 제안

### 활용 결과
- Todo List 핵심 기능 구조 도출
- EXP 및 레벨 시스템 도출
- 기린 캐릭터 성장 UI 초안 생성
- 중요도 기반 카드 강조 기능 구체화

---

## 2. 김민지

### 사용 프롬프트
- 날짜를 기준으로 매일 다른 명언을 보여주는 카드 구현
- 통계 페이지 구현
- 1주일, 1개월, 1년 단위 그래프 제공
- 완료율, 진행 현황, 레벨, 업적, 연속 수행일 분석 기능 구현

### 활용 결과
- 오늘의 명언 카드 아이디어 도출
- 통계 페이지 구조 도출
- 활동 그래프 및 성장 통계 구성
- 업적과 연속 수행일 분석 기능 구체화

---

## 3. 김도연

### 사용 프롬프트
- 과제별 집중 시간 기록 기능 추가
- 집중 시간에 따라 상태 배지가 달라지는 기능 구현
- 학교 관련 빠른 링크 카드 구현
- Classum, UWINS, 학사/장학 공지, 도서관 좌석 예약 링크 제공
- 링크 커스텀 및 편집 기능 제안

### 활용 결과
- 집중 시간 타이머 기능 구현 방향 도출
- 집중 시간 상태 배지 기준 정리
- 빠른 링크 카드 UI 구성
- 학교 서비스 연결 기능 구체화

---

## 4. 최종 정리

생성형 AI가 제안한 결과를 그대로 사용하지 않고, 발표 기간과 구현 안정성을 고려하여 핵심 기능 중심으로 통합·수정했다.
