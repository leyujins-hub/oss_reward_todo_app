# 발표 구성

## 1. 배경 및 동기
대학생은 과제, 시험, 팀플 일정이 쌓이면서 할 일을 미루거나 잊기 쉽다. 기존 Todo List는 기록 기능은 제공하지만 지속적인 사용 동기가 약하다.

## 2. Adaptiveness 설명
사용자의 마감일, 중요도, 완료 여부, 집중 시간에 따라 UI와 피드백이 달라지는 것을 Adaptiveness로 정의했다.

## 3. 프로젝트 컨셉
과제를 완료할수록 기린 캐릭터가 성장하는 보상형 Todo List 앱이다.

## 4. 주요 기능
- Todo List 기본 기능
- 캐릭터 성장 및 EXP
- 중요도 기반 카드 강조
- 날짜별 퀘스트 분류
- 집중 시간 타이머
- 통계 페이지
- 빠른 링크

## 5. GitHub 협업 과정
- 기능별 브랜치 생성
- Pull Request 활용
- 의미 있는 커밋 메시지 작성
- main branch에 최종 결과 정리
- gh-pages branch로 배포

## 6. Demo

- 집중 시간 타이머 실행
