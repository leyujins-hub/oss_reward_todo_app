# 최종 구현 기능 요약

## 1. Todo List 기본 기능

- 할 일 추가
- 할 일 조회
- 할 일 수정
- 할 일 삭제
- 진행 전, 진행 중, 완료 상태 구분
- 마감일, 중요도, 카테고리 설정

## 2. 보상형 성장 시스템

- 과제 완료 시 EXP 증가
- 현재 레벨 표시
- 다음 레벨까지 필요한 EXP 표시
- 기린 캐릭터 상태 카드 표시
- 오늘의 진행도 표시

## 3. 중요도 기반 UI

- 중요도 높은 과제는 붉은 카드로 강조
- 완료된 과제는 흐리게 표시
- 상태와 중요도에 따라 카드의 시각적 우선순위가 달라짐

## 4. 날짜 기반 퀘스트 분류

- 오늘 이전 날짜의 미완료 과제는 미룬 퀘스트로 분류
- 오늘 날짜의 과제는 오늘 퀘스트로 분류
- 내일 날짜의 과제는 내일 퀘스트로 분류
- 날짜 비교 오류를 해결하기 위해 로컬 날짜 기준으로 수정

## 5. 집중 시간 타이머

- 과제별 집중 시간 측정
- 시작, 일시정지, 종료 버튼 제공
- 초 단위 실시간 카운팅
- 종료 시 집중 시간 저장
- 완료된 과제는 타이머 숨김

## 6. 통계 페이지

- 전체 퀘스트 수
- 완료 수
- 진행 중 수
- 완료율
- 주간, 월간, 연간 활동 그래프
- 성장 통계
- 업적
- 연속 수행일
- 총 집중 시간
- 완료된 과제 평균 집중 시간

## 7. 빠른 링크

- Classum 바로가기
- UWINS 바로가기
- 학사/장학 공지 바로가기
- 도서관 좌석 예약 바로가기

## 8. 데이터 저장

- localStorage를 활용하여 할 일, 상태, 집중 시간, EXP 데이터를 저장
- 새로고침 후에도 데이터 유지
