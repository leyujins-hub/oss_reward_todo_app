# 화면 구성

## 1. 홈 화면

### 주요 구성
- 상단 캐릭터 영역
- 오늘의 추천 카드
- 전체 완료율 표시
- 투두리스트 바로가기 버튼

### Adaptive 요소
- 마감일이 가까운 과제를 추천한다.
- 완료율에 따라 캐릭터 상태가 변화한다.
- 미완료 항목이 많을 경우 경고 문구를 표시한다.

## 2. 투두리스트 화면

### 주요 구성
- 전체 과제 목록
- 완료 체크박스
- 수정 버튼
- 삭제 버튼
- 위험도 색상 카드

### Adaptive 요소
- 마감일이 가까운 항목을 상단에 배치한다.
- 중요도가 높은 항목을 강조한다.
- 미완료 횟수가 많은 항목에 경고 표시를 추가한다.

## 3. 과제 추가 화면

### 주요 구성
- 과제 제목 입력창
- 마감일 입력
- 중요도 선택
- 카테고리 선택
- 저장 버튼

### Adaptive 요소
- 마감일과 중요도에 따라 위험도를 계산한다.
- 필수값이 누락되면 안내 메시지를 표시한다.

## 4. 캐릭터 성장 화면

### 주요 구성
- 캐릭터 이미지
- 경험치 표시
- 완료한 과제 수
- 성장 단계 표시

### Adaptive 요소
- 과제 완료 시 경험치가 증가한다.
- 완료율에 따라 캐릭터와 캠퍼스 상태가 변화한다.
