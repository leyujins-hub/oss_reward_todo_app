# 프로젝트 기획

## 프로젝트명
OSS Reward To-Do App

## 프로젝트 주제
울산대학교 학생들을 위한 Adaptive To-Do List 앱

## 문제 상황
대학생은 과제, 시험, 팀플, 개인 일정 등 여러 일을 동시에 관리해야 한다. 하지만 기존 To-Do List는 단순히 할 일을 나열하는 방식이기 때문에 어떤 일이 더 급한지, 어떤 일을 계속 미루고 있는지 직관적으로 알기 어렵다.

특히 과제나 시험 일정은 마감일이 가까워질수록 중요도가 높아지지만, 일반적인 To-Do List는 이러한 상황 변화를 충분히 반영하지 못한다. 또한 사용자가 반복적으로 미루는 작업을 강하게 인식시키거나, 완료 행동에 대한 보상을 제공하는 기능도 부족하다.

## 해결 방향
본 프로젝트는 사용자의 작업 상태에 따라 UI와 추천 방식이 변화하는 Adaptive To-Do List를 구현한다.

마감일, 중요도, 미완료 여부를 기준으로 작업의 우선순위를 조정하고, 위험도를 색상으로 표시한다. 또한 과제를 완료하면 캐릭터가 성장하는 보상 구조를 통해 사용자가 계속 앱을 사용하도록 유도한다.

## 핵심 차별점
- 마감일이 가까운 작업을 자동으로 상단에 배치한다.
- 중요도가 높은 작업을 색상으로 강조한다.
- 미완료 항목을 경고 문구로 표시한다.
- 완료한 과제 수에 따라 캐릭터와 캠퍼스가 성장한다.
- 울산대학교 학생의 학교생활에 맞춘 To-Do List 구조를 제공한다.

## MVP 범위
본 프로젝트의 최소 구현 범위는 다음과 같다.

- 홈 화면
- 투두리스트 화면
- 과제 추가 화면
- 캐릭터 성장 화면

## 디자인 의사 결정
- 메인 화면 구조: 리스트형
- 항목 표시 방식: 리스트형
- 위험도 표시 방식: 색상
- Adaptive 추천 위치: 팝업
- 통계 구현 범위: 카테고리별 통계
- 색상 컨셉: 초록 / 하양
- 최소 구현 범위: 전체 화면
