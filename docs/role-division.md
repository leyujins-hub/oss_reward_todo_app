# 팀원 역할 분담 및 기여 내용

## 이유진

### 역할
- GitHub 관리
- 최종 main branch 정리
- 발표 자료 구성
- 팀원 자료 취합
- 최종 사이트 점검
- 프로젝트 컨셉 정리

### 주요 작업
- 프로젝트 최종 컨셉을 보상형 성취 Todo List 앱으로 정리
- GitHub 브랜치 및 Pull Request 흐름 관리
- README, docs 문서, 발표 자료 구조 정리
- 최종 사이트에서 날짜별 퀘스트 분류 오류 확인 및 수정
- 팀원별 초안과 최종 구현본 차이 정리

## 안혜원

### 역할
- Todo List 핵심 기능 및 캐릭터 성장 시스템 담당

### 주요 작업
- 과제 추가 화면과 입력 폼 구성
- 중요도에 따른 색상 및 폰트 강조 기능 구현
- EXP 및 레벨 시스템 설계
- 기린 캐릭터 성장 시스템 기획 및 구현
- 캐릭터 UI와 진행도 시각화 구현
- 과제 완료 체크 기능 구현
- 과제 수정 및 삭제 기능 구현

### 최종 반영
- 상단 기린 캐릭터 카드
- Lv.1, EXP, 다음 레벨까지 필요한 EXP 표시
- 오늘의 진행도 UI
- 중요도 높은 과제의 붉은 카드 강조
- Todo 추가, 수정, 삭제, 완료 흐름

## 김민지

### 역할
- 오늘의 명언 기능 및 통계 페이지 담당

### 주요 작업
- 날짜 기반 오늘의 명언 카드 기획
- 카드형 UI 및 애니메이션 효과 적용
- 주간, 월간, 연간 통계 페이지 구현
- 전체 퀘스트 수, 완료 수, 진행 중 수, 완료율 계산
- 활동 그래프 구현
- 성장 통계 구현
- 업적 시스템 구현
- 연속 수행일 분석 기능 구현

### 최종 반영
- 통계 페이지
- 1주일, 1개월, 1년 탭
- 활동 그래프
- 성장 통계
- 업적 카드
- 성장 분석
- 총 집중 시간 및 평균 집중 시간 표시

## 김도연

### 역할
- 과제별 집중 시간 타이머 및 빠른 링크 기능 담당

### 주요 작업
- 과제별 집중 시간 기록 기능 기획
- 시작, 일시정지, 종료 버튼을 통한 타이머 구성
- 집중 시간 상태 배지 설계
- 학교 관련 빠른 링크 대시보드 구현
- Classum, UWINS, 학사/장학 공지, 도서관 좌석 예약 링크 구성

### 최종 반영
- 과제 카드 내 집중 시간 타이머
- 실시간 초 단위 카운팅
- 집중 시간 저장
- 통계 페이지 내 총 집중 시간 및 평균 집중 시간 반영
- 학교 관련 빠른 링크 카드
